# DEVOPS-AI COURSE — Zero to Agentic AI Developer

> **Your mission:** Go from zero programming knowledge to building real-world Agentic AI systems used by companies — through 16 hands-on projects across 12 weeks.

---

## Who This Is For

You have **no programming or AI background**. This course starts from absolute scratch and builds you up step by step. Every concept is explained in plain English. Every project is 100% complete — no gaps, no "figure it out yourself."

---

## What You Will Build

| Phase | Projects | Skills Gained |
|-------|---------|---------------|
| **Phase 1: Foundations** (Weeks 1–2) | Hello AI, Smart Notes, Ask My Doc | Python basics, API calls, RAG |
| **Phase 2: Agents & Tools** (Weeks 3–4) | Web Research Agent, File Manager, Multi-Step Agent | LangChain, tool use, agentic loops |
| **Phase 3: Memory & Context** (Weeks 5–6) | Personal Assistant, Support Bot, Knowledge Base | Vector DBs, state management, ChromaDB |
| **Phase 4: Multi-Agent Systems** (Weeks 7–8) | Research+Writer, Code Review, Data Analysis Crew | LangGraph, agent collaboration |
| **Phase 5: Real-World Org Projects** (Weeks 9–12) | HR Recruiter, Sales Pipeline, Enterprise RAG, DevOps Agent | FastAPI, production patterns, full systems |

---

## Course Structure

Each project folder contains:

```
projectN-name/
├── README.md         ← Full lesson: what, why, code explanation, quiz
├── TRANSCRIPT.md     ← Simulated coach-student learning session
├── *.py              ← Complete, working Python code
├── requirements.txt  ← All dependencies to install
└── .env.example      ← Template for your API keys
```
