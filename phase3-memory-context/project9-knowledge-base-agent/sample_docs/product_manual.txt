SECTION 3: GETTING STARTED

3.1 Initial Setup
Connect the 65W USB-C power adapter to the port labeled "PWR" on the rear of the device. Connect your monitor using the included HDMI adapter or directly via USB-C. Connect a keyboard and mouse via USB or Bluetooth. Press the power button (circular button on the top of the device). The setup wizard will guide you through language selection, Wi-Fi configuration, and TechCorp account login.

3.2 TechCorp Account Setup
The X1 requires a TechCorp account (free) for initial setup. If you don't have an account, visit account.techcorp.com to create one. Your TechCorp account connects to TechCorp Cloud, enabling automatic backup of your documents and settings. Premium TechCorp Cloud subscribers (TechCorp Cloud Pro, $9.99/month) receive 2TB of storage and priority sync.

3.3 Connecting to TechCorp Suite
TechCorp Suite productivity software is pre-installed on the X1. Launch it from the application menu. Sign in with your TechCorp account credentials. Suite applications include: TechDoc (word processor), TechSheet (spreadsheet), TechSlide (presentations), TechMail (email client), and TechCalendar.

SECTION 4: COMMON TROUBLESHOOTING

4.1 Device Won't Power On
Check that the power adapter is properly connected to both the device and the wall outlet. Verify the wall outlet is working (test with another device). Ensure you are using the included 65W adapter — lower wattage adapters may not provide sufficient power. Hold the power button for 10 seconds to perform a forced restart. If the device still does not power on, contact TechCorp Support.

4.2 Display Issues
If no image appears on the monitor, check all cable connections. Try a different cable or port. Verify the monitor is set to the correct input source. The X1 supports up to 4K at 60Hz — if your monitor shows a "signal out of range" error, connect a keyboard and press Ctrl+Alt+F2 to reset to 1080p mode. For multi-monitor setups, connect monitors before powering on the X1.

4.3 Wi-Fi Connectivity Problems
Navigate to Settings > Network to view available networks. If Wi-Fi is not detecting any networks, try toggling Wi-Fi off and on. Check that your router is operating on a supported frequency (2.4 GHz, 5 GHz, or 6 GHz). Factory reset the network settings from Settings > Network > Reset Network Settings. If problems persist, use the Ethernet adapter (sold separately) as a temporary workaround.

4.4 Performance Issues
If the device feels slow, check running applications in the Task Manager (Ctrl+Shift+Esc). Close unused applications. If storage is above 90% full, delete or move files to TechCorp Cloud. Run the TechCorp Optimizer from Settings > System > Optimize. A full system restart often resolves temporary performance issues.
